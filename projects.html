Desktop application developed in Python designed to translate Salesforce metadata using OpenAI's API. Through a simple user interface, this tool automates the process of translating a Salesforce Organization's content.

Ecomods

2023

Ecomods is a web application that simulates an online store focused on selling modular devices. Developed with Django for the backend, it allows users to log in, browse, add to cart, and build custom modular tech products. The platform also offers login and signup options through Google authentication as well as the standard method using email and password.

2022

This project is a recreation of the popular Angry Birds game, developed entirely in Java with no additional libraries. It involves implementing the physics-based gameplay where players launch birds to destroy structures, login and user administration through tokens and level loading.
